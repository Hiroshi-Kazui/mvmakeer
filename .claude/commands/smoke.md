# Smoke Test

`app/` が存在する場合、以下を順に実行して結果を報告する:

1. `app/` で `npx tsc --noEmit`(型チェック)
2. `app/` で `npx vitest run`(単体テスト。テストが未整備の段階ならスキップし、その旨を報告)
3. `app/src-tauri/` で `cargo check`(Rust検査)

## ルール

- 失敗した場合は原因を修正してから同じチェックを再実行する。修正が設計判断を伴う場合は停止してユーザーに報告
- 全チェック通過なら「PASS」として各チェックの所要時間とともに簡潔に報告
- `app/` が未作成(Phase 0 前)の場合はその旨を報告して終了
